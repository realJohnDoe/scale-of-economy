# How Many People

A small web app to visualize and compare the sizes of various organizations, from small groups to entire countries. The application allows zooming through different orders of magnitude, up to a billion people, to provide a clear and intuitive representation of scale.

## Features
- Interactive visualization of different group sizes.
- Smooth zooming and camera transitions using CSS.
- Component-based architecture using React and TypeScript.

## Tech Stack
- Vite
- React
- TypeScript
- CSS for animations
